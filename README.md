# Heart Disease Detection 🫀

A Machine Learning project developed to predict the likelihood of heart disease in patients using clinical data. This project utilizes a **Random Forest Classifier** for high-accuracy predictions.

## 📌 Project Overview
This project focuses on identifying risk factors for heart disease. By analyzing data like age, cholesterol levels, and heart rate, the model provides a diagnostic prediction that can assist healthcare providers in early intervention.

### Key Features:
- **Data Analysis:** Visualizing heart disease trends across different age groups.
- **Model Training:** Using the Random Forest algorithm for robust performance.
- **Serialization:** Saving the model as a `.pkl` file for real-time predictions.

## 📂 Repository Structure
- `heartdiseasedataset.ipynb`: The main Google Colab notebook with the source code.
- `HeartDiseaseTrain-Test.csv`: The dataset used for training and testing.
- `heart_disease_model.pkl`: The saved Random Forest model.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn, Matplotlib
- **Environment:** Google Colab

## 🚀 How to Use
1. Clone the repo to your local machine.
2. Open `heartdiseasedataset.ipynb` in Colab or Jupyter.
3. Ensure the `.csv` file is in the same directory.
4. Run all cells to see the results.
